## Repository Guidelines

### 项目概览

这是一个 C 语言学习项目。每个 `.c` 源文件演示一个独立主题，同名的 `.md` 文件（如有）是该主题的参考手册。编译产物 `*.out` 由 `.gitignore` 屏蔽。

### 项目结构

```
.
├── hello.c              # 基础入门：printf 占位符、变量声明、赋值
├── 占位符.md            # hello.c 对应的占位符参考手册
├── operator.c           # 算术运算符、赋值简写、自增自减
├── relatiionalOperator.c # 关系运算符（注意文件名的拼写）
├── logicalOperator.c    # 逻辑运算符（短路求值等）
├── bitwiseOperator.c    # 位运算符
├── 运算符.md            # 以上四个 .c 对应的综合运算符手册
├── CLAUDE.md            # Claude Code 工作指引
├── AI-md-guide.md       # .md 参考手册编辑规范（被 .gitignore 屏蔽）
└── COPYING              # GPL 许可证
```

### 编译与运行

```bash
gcc -std=gnu23 -g <file>.c -o <file>.out
./<file>.out
```

- 编译器：GCC，标准：C23（`-std=c23`），调试符号：`-g`
- 编译产物 `*.out` 已在 `.gitignore` 中忽略
- 当前无 Makefile 或构建脚本，直接对单个源文件编译运行即可

### 代码风格

- **缩进**：4 空格
- **编码**：UTF-8，源文件中使用中文注释和字符串
- **主函数**：`int main(void)`，返回值使用 `return 0` 或 `return EXIT_SUCCESS`
- **头文件**：`#include<stdio.h>`（无空格）
- **变量声明**：统一放在函数体开头，支持 C23 的 `int y = {0}` 列表初始化语法
- 目前未使用格式化工具或 linter

### .md 参考手册编辑规范

详见 `AI-md-guide.md`（被 `.gitignore` 屏蔽，核心规则已内联在 `CLAUDE.md` 中）。要点：
- 禁止 GFM 表格，用 `-` 无序列表代替
- 代码示例用围栏代码块 ` ```c ``` `
- 每个板块末尾加 `> ⚠️ 注意事项` 引用块
- 注意事项内容须来源于对应 `.c` 源文件的演示和注释，不凭空编造
- 保留原文所有内容，只做排版增强
- 标题层级：`#` 手册名 → `##` 大板块 → `###` 子板块

### 测试

本项目无自动化测试框架。验证方式为编译后运行，确认输出符合预期。编译命令本身（无警告无错误）即为基本正确性检查。

### 提交规范

- 提交信息使用中文，描述本次学习或修改的具体内容
- 多个相关改动可在一条提交中列出，用空格分隔
- 示例：`学习了关系运算符和逻辑运算符 更新了运算符手册（运算符.md） 更新了.gitignore`

### 注意事项

- **不要主动修复 .c 源文件的编译错误**：用户正在学习，编译错误应由用户自己发现并修复
- `AI-md-guide.md` 被 `.gitignore` 屏蔽，如需查阅直接用 `cat` 读取
- 新增 `.c` 源文件时，如需对应的 `.md` 参考手册，按上述编辑规范创建
