# C 语言运算符参考手册

## 一、算术运算符

算术运算符专门用于算术运算，主要有下面几种。

- `+`：正值运算符（一元运算符）
- `-`：负值运算符（一元运算符）
- `+`：加法运算符（二元运算符）
- `-`：减法运算符（二元运算符）
- `*`：乘法运算符
- `/`：除法运算符
- `%`：余值运算符

> **⚠️ 注意事项**
>
> 1. 正值运算符 `+` 通常可以省略，`+x` 等价于 `x`。
> 2. 负值运算符 `-` 可以用于改变数值的正负：
>
> ```c
> int x = -114;
> ```
>
> 3. 这些运算符可以部分叠加使用：
>
> ```c
> int z = x + -y;
> ```
>
> 4. **除法运算**：两个整数相除时，默认进行**整数运算**，小数部分会被截断（不是四舍五入）。只有当至少一方是浮点数时，才会进行浮点运算。如需得到浮点结果，可以将操作数强制转换：
>
> ```c
> float a = 5 / 2;            // a = 2.0（整数运算截断）
> printf("%f\n", num / 2.0);  // 浮点运算（一方是浮点数）
> printf("%f\n", (float)(num / 2));  // 注意：强制转换在整数除法之后，结果仍是 2.000000
> ```
>
> 5. **整数除法的陷阱**：
>
> ```c
> num = (num / 20) * 100;
> // 由于 num / 20 进行的是整数运算，若 num < 20，得出 0
> // 而 0 乘任何数都得 0，因此最终 num = 0
> ```
>
> 6. **余值运算符 `%`**：结果的符号取决于**前一个数字**（被除数）：
>
> ```c
> int a = 6 % 4;   // a = 2
> int b = -6 % 4;  // b = -2（符号取决于被除数）
> ```

## 二、赋值运算的简写形式

如果变量对自身的值进行算术运算，C 语言提供了简写形式，允许将赋值运算符和算术运算符结合成一个运算符。

- `+=`
- `-=`
- `*=`
- `/=`
- `%=`

下面是一些例子。

```c
i += 3;  // 等同于 i = i + 3
i -= 8;  // 等同于 i = i - 8
i *= 9;  // 等同于 i = i * 9
i /= 2;  // 等同于 i = i / 2
i %= 5;  // 等同于 i = i % 5
```

> **⚠️ 注意事项**
>
> 1. 这些简写形式是 C 语言的语法糖，功能上完全等价，选择哪种写法取决于可读性偏好。
> 2. 简写形式同样遵循对应算术运算符的运算规则（如 `/=` 也受整数除法规则影响）。

## 三、自增自减运算符

- `++`：自增运算符（变量值加 1）
- `--`：自减运算符（变量值减 1）

自增/自减运算符分前缀和后缀两种形式：

- 后缀 `i++`：先返回 `i` 的原始值，再对 `i` 加 1
- 前缀 `++i`：先对 `i` 加 1，再返回新值

> **⚠️ 注意事项**
>
> 1. 前缀和后缀形式虽然有效，但在复杂表达式中容易**混淆**，难以准确判断各变量的实际值：
>
> ```c
> // 容易混淆的写法
> j = (i++ + 10);
> // 先返回 i 原始数值，然后 i+1，再进行运算
> // i 和 j 最终各是多少？需要停下来想
>
> j = (++i + 10);
> // 先自增再返回数值，再进行运算
> ```
>
> 2. **推荐写法**：将自增/自减操作与使用该变量的表达式**分开写**，意图更清晰：
>
> ```c
> // 需要先用原值再自增时
> i = 42;
> j = (i + 10);
> i++;
> printf("%i , %i\n", i, j);
>
> // 需要先自增再用新值时
> i = 42;
> i++;
> j = (i + 10);
> printf("%i , %i\n", i, j);
> ```

## 四、关系运算符

C 语言用于比较的表达式，称为"关系表达式"（relational expression），里面使用的运算符就称为"关系运算符"（relational operator），主要有下面 6 个。

- `>`：大于运算符
- `<`：小于运算符
- `>=`：大于等于运算符
- `<=`：小于等于运算符
- `==`：相等运算符
- `!=`：不相等运算符

下面是一些例子。

```c
a == b;
a != b;
a < b;
a > b;
a <= b;
a >= b;
```

关系表达式通常返回 `0` 或 `1`，表示真伪。C 语言中，`0` 表示伪，所有非零值表示真。比如，`20 > 12` 返回 `1`，`12 > 20` 返回 `0`。

关系表达式常用于 `if` 或 `while` 结构。

> **⚠️ 注意事项**
>
> 1. **`==` 与 `=` 混淆**：相等运算符 `==` 与赋值运算符 `=` 是两个不一样的运算符，不要混淆。有时可能不小心写出下面的代码，它可以运行，但很容易出现意料之外的结果：
>
> ```c
> if (x = 3)  // 原意是 x == 3，不小心写成了 x = 3
> ```
>
> 上面示例中，`x = 3` 表示对变量 `x` 赋值 `3`，返回值为 `3`（非零即为真），所以 `if` 判断总是为真。这是编译时不会报错的逻辑错误。
>
> 2. **多个关系运算符不宜连用**：
>
> ```c
> i < j < k
> ```
>
> 这是合法表达式，不会报错，但是通常达不到想要的结果（即保证变量 `j` 的值在 `i` 和 `k` 之间）。因为关系运算符是从左到右计算，实际执行的是：
>
> ```c
> (i < j) < k
> ```
>
> `i < j` 返回 `0` 或 `1`，最终是 `0` 或 `1` 与变量 `k` 进行比较。想要判断变量 `b` 的值是否在 `a` 和 `c` 之间，应该使用逻辑运算符：
>
> ```c
> if (a < b && b < c)
> {
>     printf("b 的大小在 a 和 c 之间\n");
> }
> ```

## 五、逻辑运算符

逻辑运算符提供逻辑判断功能，用于构建更复杂的表达式，主要有下面三个运算符。

- `!`：否运算符（改变单个表达式的真伪，取反）
- `&&`：与运算符（两侧的表达式都为真，则为真；否则为伪）
- `||`：或运算符（两侧至少有一个表达式为真，则为真；否则为伪）

> **⚠️ 注意事项**
>
> 1. **否运算符 `!` 的行为**：`!x` 中，当 `x` 非零时返回 `0`，`x` 为零时返回 `1`。`!` 优先级较高。
>
> ```c
> if (!(x > 2))  // 当 x <= 2 时为真
> // 更合理的做法是直接写成：
> if (x <= 2)
> ```
>
> 2. **常用用法**：
>
> ```c
> // 判断变量是否为零或未触发
> if (!data_loading) { ... }
>
> // 判断指针是否为空（同样用 !）
>
> // 使用双重否 !! 使判断严格归一化为 1 或 0
> int notZeroData = 114;
> if (!!notZeroData)  // 结果为 1
> {
>     ...
> }
> ```
>
> 3. **短路求值（Short-circuit Evaluation）**：逻辑运算符的执行顺序是**先左后右**。如果左侧运算足以确认整个表达式的真伪，**右侧不会执行**：
>
> ```c
> if (x == 2 && y == 114)
> // 左侧 x == 2 已经判断为伪，右侧 y == 114 不会执行
>
> if (x == 1 || y == 3)
> // 左侧 x == 1 已经判断为真，右侧 y == 3 不会执行
> ```
>
> 4. **短路求值带来的副作用陷阱**：由于右侧可能不被执行，应避免在逻辑表达式两侧嵌入自增/自减等有副作用的操作：
>
> ```c
> // 有问题的代码：
> while ((x++ < 10) && (x + y < 20))
> // 执行左侧表达式后，变量 x 的值就已经变了
> // 等到执行右侧表达式时，用的是新的 x 值，通常不是原始意图
> ```
>
> 5. **优先级提示**：`!` 优先级高于 `&&`，`&&` 优先级高于 `||`。混合使用时建议**加括号**明确意图。

## 六、位运算符

位运算符（bitwise operator）直接对整数的二进制位进行操作，主要有下面几种。

- `~`：取反运算符（一元运算符，将每个二进制位反转：0 变 1，1 变 0）
- `&`：与运算符（二元运算符，对应位都为 1 时返回 1，否则返回 0）
- `|`：或运算符（二元运算符，对应位有一个为 1 就返回 1，否则返回 0）

### 1. 取反运算符 `~`

`~` 是一元运算符，将操作数的每个二进制位反转。

```c
int x = 114;
printf("%i\n", ~x);  // ~x 一般等于 -x - 1
```

> **⚠️ 注意事项**
>
> 1. `~` 运算符**不会改变变量本身的值**，只是返回反转后的新值。
> 2. 对整数取反，一般情况下 `~x = -x - 1`（补码系统下）。
> 3. **进制陷阱**：C 语言默认不支持直接写二进制字面量，只支持十进制、八进制（`0` 开头）、十六进制（`0x` 开头）。以 `0` 开头的常量会被解析为**八进制数**：
>
> ```c
> printf("%i\n", ~0110001001);
> // 0110001001 被解析为八进制数，再取反
> // 结果不是直觉上二进制反转的值
> ```
>
> 4. **GNU 扩展**：使用 `0b` 前缀可以写二进制字面量，但这是 GNU C 的扩展，不是标准 C：
>
> ```c
> printf("%i\n", ~0b0110001001);  // GNU 扩展，非标准 C
> ```
>
> 5. **无符号取反**：如果需要对无符号整数进行完整位反转，应使用 `unsigned int`：
>
> ```c
> unsigned int y = 0b0110001001;  // 0b 字面量值为 393
> printf("%u\n", ~y);             // ~y = UINT_MAX - 393
> ```
>
> 6. **不能对浮点数取反**：C 语言禁止对 `float`/`double` 使用 `~` 运算符，编译时会报错。

### 2. 与运算符 `&`

`&` 将两个值的每个二进制位逐一比较：两个位都为 1 时对应结果位为 1，否则为 0。

```c
int result = 0b10010011 & 0b00111101;
// 10010011
// 00111101  (&)
// --------
// 00010001 → 十进制 17
printf("%i\n", result);
```

> **⚠️ 注意事项**
>
> 1. `&` 可以和赋值运算符简写为 `&=`：
>
> ```c
> int value = 3;
> value &= 0377;       // 等同于 value = value & 0377
> printf("%i\n", value);
> ```
>
> 2. 进制前缀规则同取反运算符：`0b` 是 GNU 扩展，`0` 开头为八进制，注意不要混淆进制的实际数值。

### 3. 或运算符 `|`

`|` 同样逐位比较两个值的二进制位：两边**至少有一个为 1** 时对应结果位为 1，两边都为 0 时结果位为 0。

> **⚠️ 注意事项**
>
> 1. `|` 也可以和赋值运算符简写为 `|=`：
>
> ```c
> value |= mask;  // 等同于 value = value | mask
> ```
>
> 2. 其余行为（进制前缀、运算规则、简写形式）参考与运算符 `&`。
